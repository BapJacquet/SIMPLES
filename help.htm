LIREC

LIREC est une plate-forme web d'écriture de documents en Facile à Lire et à Comprendre.

LIREC veut dire Lire et Ecrire.

Projet

LIREC est un outil créé dans le projet SIMPLES.

SIMPLES cherche à simplifier les documents de manière automatique, quand c'est possible.

SIMPLES est un projet soutenu par Systran, le laboratoire LUTIN-Userlab, et l'EPNAK.

Ressources

Nous utilisons le dictionnaire l'internaute pour proposer des définitions.

Nous utilisons la base de donnée Lexique3 pour trouver les mots peu fréquents.

Nous utilisons la base de donnée ARASAAC pour les pictogrammes, et aussi l'API de QWANT.
